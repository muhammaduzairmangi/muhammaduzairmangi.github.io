An ambitious World History Knowledge Platform with a RAG pipeline using local embeddings (sentence-transformers), MongoDB vector search, and 25+ ingested history books. Designed to avoid external API dependencies and support academic research.

A full-stack Flutter & Firebase app for artists and event organizers to book, manage, and make in-app payments. Features three dashboards (artist, event-organizer, admin), a ML-based recommendation system for top-rated singers, artist face recognition, and Stripe payment gateway integration.

Tech: Flutter, Firebase, Machine Learning, Stripe
